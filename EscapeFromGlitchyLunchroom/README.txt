Escape from the Glitchy Lunchroom - AP CSA Final Project

INTERACTIVE GUI GAME - click buttons to play (not typing in console).

BlueJ:
1. Open this folder as a project
2. Compile
3. Right-click Game -> main
4. Click START GAME, then ATTACK / MOVE / HEAL / CODE PATCH
5. Watch sprites and health bars update
6. Yellow tile shows where you are on the cafeteria floor

Needs images folder with all png files.

Read CODE_GUIDE.txt for what each file does.
